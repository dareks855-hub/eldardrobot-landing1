# Eldardrobot Landing — Будь Сильным 💪

Одностраничный промо-лендинг о мотивации, силе и здоровом образе жизни в духе Эльдара Дробота.

## 🚀 Стек технологий

- **Next.js 14** — современный React фреймворк
- **TypeScript** — типобезопасность
- **Tailwind CSS** — стилизация
- **Framer Motion** — гладкие анимации
- **Lucide React** — иконки

## 📋 Функциональность

- ✨ Мощный герой-экран с анимациями
- 💪 Секция с 5 принципами успеха (glassmorphism карточки)
- 📝 Интерактивный опросник из 5 вопросов
- 📊 Анимированный прогресс-бар
- 🎯 Результаты с юмористическим выводом
- 📱 Полная адаптивность под мобильные устройства
- 🌙 Тёмная тема по умолчанию

## 🛠️ Установка и запуск

```bash
# 1. Перейти в папку проекта
cd eldardrobot-landing

# 2. Установить зависимости
npm install

# 3. Запустить dev сервер
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
